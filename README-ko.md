심지어 Synthesia_X는 두가지의 알고리즘이 있습니다.

### 첫번째 알고리즘 - "AltCrack"

AltCrack는 크랙을 더 빨리 해주고 크랙에 사용한 임시 파일들을 청소합니다.

또한, 코드를 최적화해주죠!

### 두번째 알고리즘 - "DrCode"

DrCode는 'PuryStudio'에서 'band64-pixel'이 제작하였습니다.

DrCode는 Synthesia를 쉽고 간단하게 크랙 해줍니다.
